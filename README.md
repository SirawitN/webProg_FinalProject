# MicroShopee

## A term project website for **Web Development by Ruby on Rails (2110497) Academic year 2021**

<br>

### Getting started

- install Ruby
- install MySQL
- install Node.js
- install Yarn
- `gem install rails` to install Rails
- `bundle install` to install all gem files
- `rails s` to start the server
